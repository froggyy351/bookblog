## このアプリについて

- コーディング学習のために作成した書籍管理アプリ
- 自分が読んだ技術書を管理するアプリを作れば、コンテンツもアプリもポートフォリオとして使えると思い作成


## 自分ルール

- バイブコーディング禁止
- 不明点は基本的にインターネットで検索する</br>（課題に対して自分の頭で考える時間を増やすため。考える時間が長いほど長期記憶となり自分の血肉となる気がしたので）
- それでも分からなければ生成AIに聞く
- 今(2026/4)の自分のスキル感では完全に0から開発は難しい、かつ学習効率が悪いと判断し、成果物イメージを伝えたClaude(Sonnet4.6)に作成してもらったWBSをインプットに作業した。
- 実際のWBSは以下。</br>
https://docs.google.com/spreadsheets/d/1PlE-uX0UGPcoRVLaTrBPaoZDxgI2JEBiQIkO4o8dWPs/edit?usp=sharing
- またWBSを生成したコンテキストを持つチャットにPM役を依頼。以下をお願いした。

```text
・毎日スタンドアップを実施（音声チャットは精度が微妙だったので、私の音声をAqua Voiceで文字起こし）
・PMとして私の進捗にコメント。理解度や進捗に対してツッコミを適宜入れる（実務を想定）
```
